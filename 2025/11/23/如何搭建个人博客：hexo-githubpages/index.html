如何搭建个人博客：hexo + githubpages

没问题！既然你说是初学者，那我们就把所有”默认你懂”的步骤都拆解开，手把手带你走一遍。

这个方案的目标是：在 30 分钟内，让你的博客在互联网上可以通过链接访问。

我们采用 Hexo (博客框架) + GitHub Pages (免费服务器) 的组合。

第一阶段：准备工作 (环境搭建)

就像盖房子前要准备砖头和水泥，你需要先安装两个程序员必备工具。

  1. 安装 Node.js (为了运行 Hexo)

    • 去官网 nodejs.org 下载 LTS (长期支持版)。
    • 一路点击 “Next” 安装即可。
    • 验证： 打开你电脑的终端（Windows 按 Win+R 输入 cmd，Mac 打开 Terminal），输入 node -v。如果出现版本号（比如 v18.x.x），说明安装成功。

  2. 安装 Git (为了把代码传到 GitHub)

    • 去官网 git-scm.com 下载对应版本安装。
    • 验证： 在终端输入 git --version，出现版本号即成功。

  3. 注册 GitHub 账号

    • 去 github.com 注册一个账号。
    • 重要提示： 记住你的 Username (用户名)，后面非常关键。

给软工新手的特别建议

  • 不要害怕 _config.yml： 那个配置文件里有很多设置（比如网站标题、作者名字、语言），你可以大胆去改，改完执行 hexo s 看效果。记得：YAML 格式非常严格，冒号后面必须有一个空格，否则会报错。
  • 关于美化： 默认的主题叫 Landscape。如果你以后想换皮肤（主题），可以去搜 “Hexo Theme NexT” 或 “Hexo Theme Butterfly”，按照它们的文档把主题文件夹下载下来并在配置文件里改一行代码就行。
  • 遇到报错怎么办？ 复制报错信息的最后一行，直接扔给 Google 或者问我，99% 的问题前人都踩过坑。
